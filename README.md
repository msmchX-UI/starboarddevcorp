# Starboard Starter (GitHub Pages)

This is an editable static website starter for `msmchX-UI/starboarddevcorp`.

## Live URL

After enabling GitHub Pages from **Settings → Pages**:

- `https://msmchx-ui.github.io/starboarddevcorp/`

## Project Structure

- `index.html` — page content and sections
- `styles.css` — design, spacing, colors, responsiveness
- `script.js` — small interactions (theme toggle, dynamic year)

## How to Edit

1. Open the repository files on GitHub.
2. Click the pencil icon on a file.
3. Update content and commit changes.
4. GitHub Pages auto-deploys your update in ~1–2 minutes.

## Enable GitHub Pages

1. Go to repository **Settings**.
2. Open **Pages**.
3. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/ (root)`
4. Save.

## Change Branding

Edit these places:

- Site title in `index.html` (`<title>`)
- Brand text in header/footer (`Starboard`)
- Colors in `styles.css` under `:root`

## Add Custom Domain Later

When you buy/choose your domain:

1. Add your domain in **Settings → Pages → Custom domain**.
2. Configure DNS at your domain provider.
3. Enable HTTPS in GitHub Pages settings.

For `www`, create CNAME to:

- `msmchx-ui.github.io`
